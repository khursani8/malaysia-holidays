# Malaysia Holidays - Static Web App

Progressive Web App for tracking Malaysian public holidays, school holidays, and long weekend planning.

## Live Site

Visit: https://khursani8.github.io/malaysia-holidays/

## Features

- **Holiday Calendar** - Complete database of Malaysian public holidays (2025-2028)
- **Long Weekend Calculator** - Automatically identifies extended weekend opportunities
- **Multi-State Support** - All 16 Malaysian states + federal territories
- **School Holiday Tracker** - Official school holidays and exam dates
- **Personal Analytics** - Track holiday patterns and statistics
- **Weather Integration** - Contextual weather for planning

## Notes

- This is a **static-only build** running in localStorage mode
- All personal data is stored locally in the browser
